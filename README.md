# 🏅 Sports Mart Management System

A Graphics-based Sports Mart Management System developed in C using the BGI Graphics Library. This project was built for my Programming for Problem Solving (PPS) coursework. It simulates a sports equipment store where users can interact through a graphical interface, select products using the mouse, generate bills, and save purchase records to a file.


## ✨ Features
- 🎨 Interactive GUI using `graphics.h`
- 🖱️ Mouse-based item selection
- 👤 Customer name input
- 🛍️ Shopping cart with quantity management
- 🧾 Automatic bill generation
- ✏️ Edit order and Cancel order options
- 💳 Payment confirmation
- 💾 Persistent storage (stores customer details and bills in `mart.txt`)

## 🛠️ Technologies & Concepts Used
- **Language:** C / C++ 
- **Library:** BGI Graphics (`graphics.h`)
- **Core Concepts:** 
  - File Handling
  - Pointers & Arrays
  - Structures (`struct`)
  - User-defined Functions
  - Mouse and Keyboard Event-driven Programming
